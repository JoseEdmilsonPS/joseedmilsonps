Visualização de Dados

Usando ggplt2 no R

Muito importante na análise de dados é o poder dos gráficos, saber interpretá-los será de fundamental importância para se tirar os melhores insights - valor obtido com as análises, informações de valor reais que possibilitam novos negócios e melhores tomada de decisão. Não seria um exagero dizer que é uma arte a ser desenvolvida pelos profissionais que trabalham com dados.

Neste artigo buscamos mostrar de maneira bem simples alguns tipos de gráficos desenvolvidos com o R utilizando o pacote ggplt2. Não temos a pretensão inicial das análises dos dados, mas sim mostrar a importância da visualização dos dados. Se precisar de mais informações e como construir os gráficos estará disponível o material completo no site www.mundoestatistico.github.io. Será utilizado o dadaset Melanoma contido no pacote MASS do R.

1. Gráfico de dispersão

Neste tipo de gráfico as variáveis devem ser numéricas. O gráfico de dispersão é muito útil para verificar a relação entre duas variáveis, se existe alguma tendência entre ambas.

2. Gráfico boxplot

O boxplot é um dos gráficos mais úteis para visualização da distribuição de dados numéricos. Neste tipo de gráficos podemos verificar a ocorrência de outliers, a distribuição dos dados, a simetria dos dados, a mediana, o primeiro e o terceiro quartis dos dados. Uma das variáveis deve ser numérica, é mais usado para comparar a distribuição de vários grupos. No entanto, você deve ter em mente que a distribuição de dados está oculta atrás de cada caixa.

3. Gráfico de barra

O gráfico de barra é um muito útil para visualização resumida de um conjunto de dados categóricos. Cada barra no gráfico representa uma categoria particular com sua frequência de dados. Um dos gráficos mais fáceis de interpretar. Podem ser utilizadas na vertical, também conhecido como gráfico de colunas, ou na horizontal.

4. Gráfico histograma

Muito útil para verificar se a variável tem distribuição normal, procedimento inicial para um modelo de machine learning, um modelo de regressão por exemplo, para aplicar um teste de hipóteses específico que necessita desse requisito. Também utilizado na detecção de outliers, ou seja, dados que podem interferir nos resultados da análise.

5. Gráfico de Setor

O gráfico de pizza (de setores) é um gráfico circular onde os valores de cada categoria representada são proporcionais às frequências (fatia da pizza). O gráfico de pizza é uma maneira de resumir a informação de uma variável qualitativa/categórica. Serve para comparar quantidades em cada categoria.

Hoje o mundo é digital, a cada instante é gerado grande quantidade de dados e por sua vez é de suma importância saber analisar estes dados transformando em informações para melhorar o mundo no qual estamos inseridos. Daí a necessidade de saber criar e interpretar estas informações de forma gráfica, aqui tentamos mostrar os principais tipos de gráficos e sua importância. A visualização de dados é uma forma de arte visual, é contar as histórias dos dados a partir de gráficos conhecendo tendências e exceções.
